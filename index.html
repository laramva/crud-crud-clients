Preencha os dados abaixo para adicionar um novo cliente. Os registros são salvos diretamente na API do CRUDCrud.

Como funciona este projeto

Este cadastro usa Fetch API para se comunicar com o CRUDCrud. Cada cliente inserido é enviado via requisição POST, listado com GET e removido com DELETE.
